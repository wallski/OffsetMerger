# OffsetsMerger

Merges `.hpp` offset files generated by a2x's CS2 dumper into a single header file, organizing all offsets into nested namespaces under:

```cpp
namespace Offsets
```
## Requirements

- https://github.com/a2x/cs2-dumper

## Usage

1. Run the dumper to generate offsets  
2. Open OffsetsMerger  
3. Select the dumper `output` folder  
4. Choose the `.hpp` files to merge  
5. Select output directory and file name  
6. Click merge

Notes

-Each class or context gets its own namespace.

-Global offsets and signatures are placed in Offsets::global.

-Duplicate names across different classes are preserved in their respective namespaces.

-No validation is performed on offset values.
